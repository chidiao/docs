## 快捷键

| 快捷键                | 功能                         |
| --------------------- | ---------------------------- |
| `ctrl` + `backspace`  | 删除一行内，光标前的所有内容 |
| `ctrl` + `x`          | 删除一整行                   |
|                       |                              |
| `alt` + `shift` + `i` | 在多行的末尾插入光标         |
| `alt` + `click`       | 手动任意位置插入光标         |
|                       |                              |
| `ctrl` + `/`          | 单行注释                     |
| `alt` + `shift` + `a` | 手动任意位置插入光标         |
