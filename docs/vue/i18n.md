# Vue I18n

## v8.x

[vue-i18n](https://github.com/kazupon/vue-i18n)

```sh
yarn add vue-i18n
```

::: code-group

```js [main.js]
import Vue from 'vue'
import VueI18n from 'vue-i18n'
Vue.use(VueI18n)

import messages from '@/locale'
const i18n = new VueI18n({
  locale: 'en',
  messages
})

new Vue({
  i18n
}).$mount('#app')
```

```js [locale]
export const zh = {
  msg: '你好世界!'
}

export const en = {
  msg: 'Hello World!'
}

export default messages = {
  zh,
  en
}
```

```html [App]
<div>{{ $t('msg') }}</div>
```

:::

## v9.x

[vue-i18n-next](https://github.com/intlify/vue-i18n-next)

```sh
yarn add vue-i18n@9
```

::: code-group

```js [main.js]
import { createApp } from 'vue'
import VueI18n from 'vue-i18n'

import messages from '@/locale'
const i18n = VueI18n.createI18n({
  locale: 'en',
  fallbackLocale: 'en',
  messages
})

const app = createApp({})
app.use(i18n).mount('#app')
```

:::
