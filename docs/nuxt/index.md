# hosts

185.199.109.133 raw.githubusercontent.com

## degit

[starter](https://github.com/nuxt/starter)

通过拉取模板来初始化项目

```sh
# degit
degit https://github.com/nuxt/starter#v3 myapp
degit https://github.com/nuxt/starter#ui myapp
```
