# package.json

## 生命周期

```json
{
  "scripts": {
    "prepare": "echo hello!"
  }
}
```

[scripts](https://docs.npmjs.com/cli/v10/using-npm/scripts)

| 周期        | 描述                                       |
| ----------- | ------------------------------------------ |
| **prepare** | 在 `npm install` 和 `npm publish` 都会触发 |
