# Method

## GET/POST

| 区别 | Get                                           | Post                                       |
| ---- | --------------------------------------------- | ------------------------------------------ |
| 作用 | 获取资源                                      | 提交资源                                   |
| 参数 | 暴露在 `URL` 中，只接受 `ASCII 字符` 数据类型 | 包含在 `request body` 中，数据类型不受限制 |
| 长度 | 2k                                            | 不受限制                                   |
| 缓存 | 浏览器主动缓存                                | 不缓存，除非手动设置                       |
